# Solution

$x*y$ = yellow + brown

$(x-2)*(y-2)$ = yellow

```py
def solution(brown, yellow):
    n = brown + yellow
    for x in range(1, n):
        if n % x == 0:
            y = n // x
            if (x - 2) * (y - 2) == yellow:
                return [y, x]


print(solution(10, 2))  # [4, 3]
print(solution(8, 1))  # [3, 3]
print(solution(24, 24))  # [8, 6]
```
