# Solution

$9 = 01001_2$

```
format(9, 'b').zfill(5)
```

```py
def solution(n, arr1, arr2):
    answer = []
    n = len(arr1)
    for i in range(n):
        tmp = ''
        s1 = format(arr1[i], 'b').zfill(n)
        s2 = format(arr2[i], 'b').zfill(n)
        for j in range(n):
            tmp += ' ' if s1[j] == s2[j] == '0' else '#'
        answer.append(tmp)
    return answer
```
